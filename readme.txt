🚀 Crime Connection Details Website 🌐
Developed a 5-page web application using Angular, HTML, CSS, TypeScript, Bootstrap, and Google Icons to display information about individuals linked to crimes. The website fetches and presents data from a dummy API, showcasing details such as name, address, and job in a structured and user-friendly format.
🔹 Tech Stack:
✅ Angular for frontend development
 ✅ Bootstrap & Google Icons for styling and UI components
 ✅ TypeScript for enhanced development experience
 ✅ Dummy API for dynamic data rendering
This project demonstrates API integration, responsive design, and structured data presentation in a professional layout. 💻✨
